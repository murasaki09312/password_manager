# Password Manager (Shell Script)

A simple command-line password manager built with Bash.
This tool allows users to store and retrieve service credentials (service name, username, and password) efficiently.

## Features

- **Add Password**: Store a new service, username, and password.
- **Get Password**: Retrieve credentials by searching for a service name.
- **Interactive Menu**: A loop-based menu system (`Add Password`, `Get Password`, `Exit`).
- **Data Persistence**: Data is saved to a local `passwords.txt` file.

## Requirement

- Linux or macOS environment (Bash shell)

## Usage

1. Clone this repository or download the script.
2. Grant execution permission to the script:
   ```bash
   chmod +x password_manager.sh
